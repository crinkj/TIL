> ## TIL의 목적 
  ### 이제까지 사용은 해왔으나 제대로 알지 못했던것들이나 새롭게 배운 개념들을 매일마다 조금씩 찾아서 공부하고 정리하기 위한 레포지토리입니다.

> ### TIL의 생성 이유
* 개발 중 `왜?` 라는 질문을 습관화 하자.
* 무지성 개발을 피하자.
* 사용을 해도 설명을 못하면 모르는것만 못하다.
